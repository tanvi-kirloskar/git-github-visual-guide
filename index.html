Git Command Categories

Category Commands
Repository Setup git init, git clone
Tracking Changes git status, git add
Saving Work git commit
Remote Repositories git push, git pull, git fetch
Branching git branch, git checkout, git merge
Advanced git stash, git reset, git rebase

Important Terms

  • Repository: Project Folder
  • Commit: Save Point
  • Branch: Parallel Development Path
  • Merge: Combine Changes
  • Remote: GitHub Repository
