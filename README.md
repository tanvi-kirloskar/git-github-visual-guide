# Git & GitHub Visual Learning Portal

A beginner-friendly interactive website designed to learn Git and GitHub visually.

## Features

* Git vs GitHub Comparison
* Visual Git Workflow Diagram
* Essential Git Commands Cheat Sheet
* Git Terminology Guide
* Hands-on Practice Exercise
* Interactive Git Command Quiz
* Responsive Dark-Themed UI

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Learning Objectives

This project was created to:

* Understand Git fundamentals
* Learn commonly used Git commands
* Practice GitHub repository management
* Gain hands-on experience with version control

## Project Structure

```text
index.html      -> Main learning portal
quiz.html       -> Interactive Git Quiz
style.css       -> Styling
script.js       -> Homepage functionality
quiz.js         -> Quiz logic
```

## Git Workflow Covered

Create File → Edit → git add → git commit → git push → GitHub
